# Sistema de Gestión de Productos en Java

## Descripción

Este proyecto fue desarrollado en Java para demostrar la implementación de los conceptos fundamentales de la Programación Orientada a Objetos (POO).

El programa incluye:

- Una clase `Product` con atributos privados.
- Constructores (por defecto y parametrizado).
- Métodos getter y setter.
- Encapsulación.
- Sobreescritura del método `toString()`.
- Una clase principal (`App`) para probar el funcionamiento de los objetos.

El proyecto simula un sistema básico de gestión de productos donde se puede almacenar y modificar información como el ID, nombre, precio y stock de un producto.
